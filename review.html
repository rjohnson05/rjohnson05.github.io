Hosts Review

HostGator

  • Cost: Starts at $2.75/month
  • Offers 1-year Domain Registration
  • Provides Website Builder
  • Website & Domain Transfers
  • 45-day Money Back Guarantee
  • One-click WordPress Installation

DreamHost

  • Cost: Starts at $1.99/month
  • Works well with WordPress
  • WordPress Website Builder
  • WordPress Website Builder

iPage

  • Cost: Starts at $1.99/month
  • Easy WordPress Install
  • Unlimited Email Addresses
  • Unlimited Domain Names
  • Website Builder

Conclusion:

After this inital research, I would choose iPage as the host for a new website. They have a lower starting price than HostGator and offer unlimited domain names. All of these hosting services seem to offer Wordpress installations and website builders. However, iPage also offers unlimited email addresses, whereas an extra cost in charged with the other two services.
